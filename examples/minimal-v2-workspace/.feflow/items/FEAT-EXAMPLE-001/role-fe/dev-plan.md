# 开发计划（v2）

- **Item ID**: FEAT-EXAMPLE-001
- **标题**: 搜索页增加最近搜索入口
- **风险级别**: L2

## 改动策略

在不改变现有搜索提交流程的前提下，复用现有 search store 增加最近搜索展示和点击回填。

## 关键文件

- `src/pages/search/index.vue` — 增加入口和列表展示
- `src/stores/search.ts` — 持久化最近搜索并提供读取方法

## 执行步骤

1. 读搜索页和 search store 的现有逻辑
2. 在 store 中增加最近搜索记录和去重逻辑
3. 在页面中渲染最近搜索列表并支持点击回填

## 验证方式

- 手动验证输入、提交、刷新和点击回填
- 运行 `pnpm vitest`

## 风险与回退

- **主要风险**: 旧 query 回填行为被误伤
- **回退方式**: 回退 store 改动并移除页面入口
